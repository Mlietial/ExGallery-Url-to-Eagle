## 中文说明

### 1. 简介

`ExGallery Url to Eagle` 是一个用户脚本，用于在 ExHentai / E-Hentai 画廊页面提取标签，并把当前画廊作为书签项目同步到 Eagle。

脚本面板标题为 `Eagle Tags Picker`。

它支持：
- 从页面提取标签
- 勾选需要同步的标签
- 手动编辑标签值
- 自定义后缀规则
- 选择 Eagle 文件夹
- 将不同命名空间的标签同步到不同的 Eagle Tag Group
- 拖拽、缩小、贴边隐藏、顶部控件折叠

如果 Eagle 中已经存在相同页面 URL 的项目，脚本会更新它，而不是重复新建。

### 2. 运行前提

需要：
- 已安装 Tampermonkey / Violentmonkey 等用户脚本管理器
- 本地已运行 Eagle
- Eagle 本地 API 可访问，默认地址为 `http://localhost:41595`

脚本使用的权限和连接：
- `GM_xmlhttpRequest`
- `GM_getValue`
- `GM_setValue`
- `localhost`
- `127.0.0.1`

### 3. 主要功能

#### 3.1 标签提取与编辑

脚本会从画廊页面读取标签，并显示在面板列表中。

你可以：
- 单独勾选或取消勾选标签
- 使用 `Select All` / `Clear All`
- 直接修改每一行标签的文本值

手动编辑过的标签值会持久化保存，下次再次打开同一标签时仍会保留。

#### 3.2 自定义后缀

面板中提供 `Suffix` 输入框。

用途：
- 对 `artist`、`character`、`parody`、`group` 之外的命名空间标签追加自定义后缀

例如：
- `male: glasses` 可变为 `glasses-C`
- 如果后缀为空，则不会自动追加后缀

说明：
- 默认后缀为空
- 由用户自行填写
- 修改后会立即刷新当前列表中的默认标签值
- 已手动改写的标签不会被自动覆盖

#### 3.3 Eagle 文件夹选择

你可以在 `Folder` 下拉框中选择目标文件夹，并用 `Folders` 按钮刷新 Eagle 文件夹列表。

同步时：
- 如果选择了文件夹，则书签项目会同步到对应文件夹
- 如果未选择，则同步到 Eagle 根目录

#### 3.4 Eagle Tag Group 同步

脚本支持将不同类型的标签同步到不同的 Eagle Tag Group。

当前支持的组别选择：
- `Suffix`
- `Parody`
- `Character`
- `Group`
- `Artist`

每一项都可以：
- 选择一个现有的 Eagle Tag Group
- 点击 `Groups` 刷新组列表

同步规则：

1. `Suffix`
   - 同步所有以当前自定义后缀结尾的标签
   - 如果后缀为空，则该项会跳过

2. `Parody`
   - 同步 `parody` 命名空间标签

3. `Character`
   - 同步 `character` 命名空间标签
   - 会结合 `parody` 标签尝试解析角色标签
   - 会优先匹配 Eagle 中现有的相关角色标签

4. `Group`
   - 同步 `group` 命名空间标签

5. `Artist`
   - 同步 `artist` 命名空间标签

兼容旧版本：
- 如果旧版本里保存过 `Creator` 分组设置，新版本会自动把它迁移到 `Group` 和 `Artist`

### 4. 面板交互

标题栏按钮从左到右分别用于：
- 折叠/展开顶部控件
- 缩小/还原面板
- 贴边隐藏/取消贴边隐藏
面板可以贴到左边或右边隐藏：
- 点击标题栏贴边按钮
- 或拖拽到屏幕左右边缘

贴边后：
- 面板会收进侧边
- 鼠标移到侧边把手时展开
- 移开后收回

右侧贴边时，脚本会尽量避开浏览器滚动条区域。

#### 4.5 拖拽与记忆

面板支持拖拽定位，并记住以下状态：
- 面板位置
- 缩小状态
- 贴边状态
- 顶部控件折叠状态

### 5. 同步逻辑

点击 `Sync Selected to Eagle` 后，脚本会：

1. 收集当前选中的标签
2. 生成当前页面标题和 URL
3. 检查 Eagle 中是否已存在相同 URL 的项目
4. 如果已存在，则更新该项目
5. 如果不存在，则创建书签项目
6. 按配置把标签加入对应的 Eagle Tag Group

说明：
- 新建项目后，脚本当前会提示你再点一次 `Sync` 以完成标题最终确认
- 已存在项目时会直接更新标签和标题

### 6. 建议使用流程

1. 打开 ExHentai / E-Hentai 画廊页面
2. 等待面板出现
3. 点击 `Refresh` 重新读取标签
4. 选择目标文件夹
5. 按需要填写 `Suffix`
6. 为 `Suffix / Parody / Character / Group / Artist` 选择对应的 Eagle Tag Group
7. 勾选或修改要同步的标签
8. 点击 `Sync Selected to Eagle`

### 7. 常见问题

请确认：
- Eagle 已启动
- 本地 API 可访问
- 默认端口没有变化
